### SQLite Admin or SQLite Viewer

* A simple SQLite Admin and SQLite Viewer for PHP developers/development.

#### Installation and Running
```bash
git clone https://github.com/SQLiteAdmin/SQLiteAdmin.git
```
```bash
cd SQLiteAdmin
```
```bash
php -S localhost:8088
```
